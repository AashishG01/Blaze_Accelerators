# Blaze_Accelerators — Day One Bank Prototype

Day One is a high-fidelity, interactive React application prototype for a first-salary banking experience built for the Blaze Hackathon.

## The Strategy

Rather than building a standard, feature-heavy neobanking menu, **Day One** focuses on a single narrative spine: **Credit Story**.
Every transaction, budget allocation, SIP start, and financial milestone contributes points in real-time to the user's Credit Story score, showing clear unlocks (credit cards at month 6, low-rate loans at month 12).

## Tech Stack

- **Framework:** React 19 + Vite
- **State Management:** React Context + `useReducer`
- **Styling:** Vanilla CSS (glassmorphic dark theme)

## Running Locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the Vite development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:5173/](http://localhost:5173/) in your browser.
